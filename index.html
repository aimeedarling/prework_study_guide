HTML

  • HTML stands for Hypertext Markup Language
  • HTML is used to creat the structure of a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • href stands for hypertext reference and attributes specific location of the external resource

CSS

  • CSS stands for Cascading Style Sheets
  • CSS adds style to our webpage, for example colors, aesthetics, visual layouds, and organization
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside the element.

Git

  • git status: checks what branch we are currently on
  • git check out -b branch-name: creates a new branch and switches to it
  • git clone (paste copied SSH value): clones github repository onto our computer
  • git add -A: adds all monifications to the next git commit
  • git commit -m"comment about changes": commits changes to repository
  • git pull origin main: pulls the latest version of the project onto your local computer
  • git push origin feature/branch-name: pushes changes made locally to our remote github branch

JavaScript

  • JavaScript (JS) is a powerful programming language that can add interactivity to a website.
  • With JS you can build games, animated 2D and 3D graphics, comprehensive databased-driven apps, and more!
  • Remember: JS is case sensitive, each line ends with a semicolon ";"
  • Variables are named containers that store value/data in our code
  • Strings: contained in quotation marks
  • Arrays are single variables that are used to hold a group of data
  • JS arrays are a zero index (the start at 0, not 1)
  • Control flow: the order in which a computer execudes code in a script. JS is read from top to bottow
  • 'for' loops have 3 elements, keywoprd 'for', three part conditonal statement, and code block
  • Functions are used in JS to define and reuse blocks of code that contain instructions to perform a specific task.
  • Functions have three parts, function keyword, a unique name followed by parentheses, then curly brackets that contain the code for the desired task.
